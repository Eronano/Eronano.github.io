题目描述

在吉伊卡哇（Chiikawa）的世界中，坏大强使用游戏机抢走了飞鼠宝宝莫莫伽的身体四处捣蛋，为了帮助莫莫伽换回身体，Eronano需要解开游戏机的谜题：

游戏机上有一个长度为 nnn 的排列 a1…ana_1\dots a_na1​…an​，但是与普通的排列不同，a1…ana_1\dots a_na1​…an​ 包含的是 0∼n−10 \sim {n - 1}0∼n−1 之间的 nnn 个整数。

狡猾的大强删除了这个排列，对于每一个元素 i (0≤i≤n−1)i~(0 \leq i \leq {n - 1})i (0≤i≤n−1) 它都用一个正整数 LiL_iLi​ 作为还原这个排列的限制：

  • 对于每一个元素 iii，在原排列中都至少有一个长度为 LiL_iLi​ 的区间 [l,r][l, r][l,r]（即 r−l+1=Lir - l + 1 = L_ir−l+1=Li​）满足 MEXk=lrpk=i{MEX}_{k=l}^r p_k = iMEXk=lr​pk​=i。

请你帮 Eronano 求出总共有多少个合法排列，答案对 998244353998244353998244353 取模。

输入格式

第一行包含一个整数 T (1≤T≤10)T~(1 \leq T \leq 10)T (1≤T≤10) 代表测试样例的数量。

对于每一个测试样例：

  • 第一行包含一个整数 n (1≤n≤5×103)n~(1 \leq n \leq 5 \times 10^3)n (1≤n≤5×103)；
  • 第二行包含 nnn 个整数 L1,L2,…,Ln (1≤Li≤n)L_1, L_2, \dots, L_n~(1 \leq L_i \leq n)L1​,L2​,…,Ln​ (1≤Li​≤n)。

输入输出样例

说明/提示

一个序列的 MEXMEXMEX 是序列中没有出现过的最小非负整数，例如 MEXMEXMEX{1,3,41, 3, 41,3,4}$ = 0，，，MEX${0,1,3,40, 1, 3, 40,1,3,4}$ = 2$。

【样例解释】

对于第一个样例，所有可能的排列为{3012，2013，3102，2103}\{3012，2013，3102，2103\}{3012，2013，3102，2103}，以下结论对每一个排列均符合。

对于L0=1L_0 = 1L0​=1，我们可以找到排列中 MEXMEXMEX{111}$ = 0，，，MEX${222}$ = 0，，，MEX${333}$ = 0$并且每一个区间的长度为 111；

对于L1=1L_1 = 1L1​=1，我们可以找到排列中 MEXMEXMEX{000}$ = 1$并且区间的长度为 111；

对于L2=3L_2 = 3L2​=3，我们可以找到排列中 MEXMEXMEX{3,0,13, 0, 13,0,1}$ = 2，，，MEX${0,1,30, 1, 30,1,3}$ = 2，，，MEX${3,1,03, 1, 03,1,0}$ = 2，，，MEX${1,0,31, 0, 31,0,3}$ = 2$并且每一个区间的长度为 333；

对于L3=3L_3 = 3L3​=3，我们可以找到排列中 MEXMEXMEX{0,1,20, 1, 20,1,2}$ = 3，，，MEX${2,0,12, 0, 12,0,1}$ = 3，，，MEX${1,0,21, 0, 21,0,2}$ = 3，，，MEX${2,1,02, 1, 02,1,0}$ = 3$并且每一个区间的长度为 333 。

Solution

Code 
#include <iostream>
#include <ctime>
#include <vector>
#define int long long
const int mod = 998244353;
using namespace std;
void solve() {
    int n;
    cin >> n;
    vector<int> L(n);
    vector<vector<int>> dp(n + 1, vector<int> (n + 1));
    for(auto &it : L) cin >> it;
    for(int i = 0; i < n; ++i){
        if(L[i] < i){
            cout << 0 << '\n';
            return;
        }
    }
    for(int pos = 1; pos <= n; ++pos){
        if(pos > L[0] || (n - pos + 1) > L[0]) dp[pos][pos] = 1;
    }
    for(int len = 2; len <= n; ++len){
        for(int l = 1; l + len - 1 <= n; ++l){
            int r = l + len - 1;
            if(n - l >= L[len - 1]) dp[l][r] += dp[l + 1][r];
			if(r - 1 >= L[len - 1]) dp[l][r] += dp[l][r - 1];
            dp[l][r] %= mod;
        }
    }
    cout << dp[1][n] << '\n';
}
//=====================================
signed main() {
#ifdef LOCAL
    freopen("in.in", "r", stdin);
    freopen("out.out", "w", stdout);
    clock_t c1 = clock();
#endif
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    //=====================================
    int t;
    cin >> t;
    while(t--) solve();
    //=====================================
#ifdef LOCAL
    cerr << "Time used: " << clock() - c1 << " ms" << '\n';
#endif
    return 0;
}
