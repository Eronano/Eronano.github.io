题目描述

在一次讨伐中坏奇美拉使用游戏机抢走了莫莫伽的身体四处捣蛋，为了帮助莫莫伽换回身体，Eronano 需要解开游戏机的谜题：

游戏机上有一个长度为 nnn 的排列 a1…ana_1 \dots a_na1​…an​，但是与普通的排列不同，a1…ana_1 \dots a_na1​…an​ 包含的是 0∼n−10 \sim {n - 1}0∼n−1 之间的 nnn 个整数。

狡猾的奇美拉删除了这个排列，对于每一个元素 i (0≤i≤n−1)i~(0 \leq i \leq {n - 1})i (0≤i≤n−1) 它都用一个正整数 LiL_iLi​ 作为还原这个排列的限制：

  • 对于每一个元素 iii，在原排列中都至少有一个长度为 LiL_iLi​ 的区间 [l,r][l, r][l,r]（即 r−l+1=Lir - l + 1 = L_ir−l+1=Li​）满足 MEXk=lrpk=i{MEX}_{k=l}^r p_k = iMEXk=lr​pk​=i。

请你帮 Eronano 求出总共有多少个合法排列，答案对 998244353998244353998244353 取模。

说明/提示

  • 长度为 nnn 的排列是由 nnn 个不同的整数组成的数组，这些整数从 111 到 nnn 依次排列。例如 [2,3,1,5,4][2,3,1,5,4][2,3,1,5,4] 是一个排列，但 [1,2,2][1,2,2][1,2,2] 不是一个排列 (2在数组中出现了两次)(2 在数组中出现了两次)(2在数组中出现了两次)，[1,3,4][1,3,4][1,3,4] 也不是一个排列 (n=3，但数组中有4)(n = 3，但数组中有 4)(n=3，但数组中有4)。

  • 一个序列的 MEXMEXMEX 是序列中没有出现过的最小非负整数，例如 MEX{1,3,4}=0MEX \{1, 3, 4 \} = 0MEX{1,3,4}=0，MEX{0,1,3,4}=2MEX \{0, 1, 3, 4 \} = 2MEX{0,1,3,4}=2。

【样例解释】

对于第一个样例，所有可能的排列为 {[3,0,1,2]，[2,0,1,3]，[3,1,0,2]，[2,1,0,3]}\{[3,0,1,2]，[2,0,1,3]，[3,1,0,2]，[2,1,0,3] \}{[3,0,1,2]，[2,0,1,3]，[3,1,0,2]，[2,1,0,3]}，以下结论对每一个排列均符合。

对于 L0=1L_0 = 1L0​=1，我们可以找到排列中 MEX{1}=0MEX \{1 \} = 0MEX{1}=0，MEX{2}=0MEX \{2 \} = 0MEX{2}=0，MEX{3}=0MEX \{3 \} = 0MEX{3}=0 并且每一个区间的长度为 111；

对于 L1=1L_1 = 1L1​=1，我们可以找到排列中 MEX{0}=1MEX \{0 \} = 1MEX{0}=1 并且区间的长度为 111；

对于 L2=3L_2 = 3L2​=3，我们可以分别对每一种排列找到 MEX{3,0,1}=2MEX \{3, 0, 1 \} = 2MEX{3,0,1}=2，MEX{0,1,3}=2MEX \{0, 1, 3 \} = 2MEX{0,1,3}=2，MEX{3,1,0}=2MEX \{3, 1, 0 \} = 2MEX{3,1,0}=2，MEX{1,0,3}=2MEX \{1, 0, 3 \} = 2MEX{1,0,3}=2 并且每一个区间的长度为 333；

Solution

  1. 从 000 至 n−1n - 1n−1 每次考虑往排列中添加一个元素，倘若当前元素为 iii ，判定排列可能存在的条件为此前必定存在一个 连续的 排列 0∼i−10 \sim {i - 1}0∼i−1 ，使得排列长度 i≤Lii \leq L_ii≤Li​ （若连续序列中存在大于 iii 的元素不会对 MEXMEXMEX 的值造成影响）；

  2. 为了保证每次添加一个元素之后当前排列的连续性，我们考虑每次只在原排列的 最左端 或 最右端 添加元素；

  3. 令 dp[l][r]dp[l][r]dp[l][r] 为当前位置从 lll 到 rrr 的排列的方案数，我们需要在初始化时（相当于往空排列中放置 000 时）考虑我们需要在元素的左右两端预留一段多长的空间，即 L0L_0L0​ ，以此保证位置的合法性；

  4. 若当前需要计算方案数的排列的左端点索引为 lll ，右端点为 rrr ，长度为 lenlenlen ,判定排列左端可以添加元素的条件为 从 l+1l + 1l+1 开始到 nnn 的长度足以容纳一段长度为 Llen−1L_{len - 1}Llen−1​ 的排列 ，判定右侧可以添加元素的条件同理。

该解法时间复杂度 O(n2)O(n^2)O(n2)

Code 
#include <iostream>
#include <ctime>
#include <vector>
#define int long long
const int mod = 998244353;
using namespace std;
void solve() {
    int n;
    cin >> n;
    vector<int> L(n);
    vector<vector<int>> dp(n + 1, vector<int> (n + 1));
    for(auto &it : L) cin >> it;
    for(int i = 0; i < n; ++i){
        if(L[i] < i){
            cout << 0 << '\n';
            return;
        }
    }
    for(int pos = 1; pos <= n; ++pos){
        if(pos > L[0] || (n - pos + 1) > L[0]) dp[pos][pos] = 1;
    }
    for(int len = 2; len <= n; ++len){
        for(int l = 1; l + len - 1 <= n; ++l){
            int r = l + len - 1;
            if(n - l >= L[len - 1]) dp[l][r] += dp[l + 1][r];
			if(r - 1 >= L[len - 1]) dp[l][r] += dp[l][r - 1];
            dp[l][r] %= mod;
        }
    }
    cout << dp[1][n] << '\n';
}
//=====================================
signed main() {
#ifdef LOCAL
    freopen("in.in", "r", stdin);
    freopen("out.out", "w", stdout);
    clock_t c1 = clock();
#endif
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    //=====================================
    int t;
    cin >> t;
    while(t--) solve();
    //=====================================
#ifdef LOCAL
    cerr << "Time used: " << clock() - c1 << " ms" << '\n';
#endif
    return 0;
}
